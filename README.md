# Life-Fit Training Planner

A static GitHub Pages app for adapting a 3-day beginner/intermediate strength program to a busy parenting schedule.

## Features

- Calculates recommended loads from squat, bench press, and deadlift 1RM values.
- Runs Day 1, Day 2, and Day 3 in sequence even when training only once per week.
- Adjusts load and volume based on days since the last workout.
- Gives concrete accessory exercise choices and estimated starting loads.
- Saves settings locally in the browser.

## GitHub Pages

The app is served from `index.html` at the repository root.
